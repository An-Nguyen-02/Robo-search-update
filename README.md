# Robo Search Update
## Descriptions:
This app is the advance version of Robo Search with testing. It provide a searcher for existed robots from https://jsonplaceholder.typicode.com/users
## How to use:
1. Clone this repo
2. Run `npm install`
3. Run `npm update`
4. Run `npm start`

## Upcoming features:
- Progessive web app.
- Server side rendering with Next.js.
